How do I install the extension?

Wonder how to use extension? Don't worry!


1. Install ZIP file

First you must do is download ZIP file using the button on the top of this site or button below.


2. Export ZIP folder

Once you have downloaded ZIP file, you have to export it - right-click the file and click "Export all".


3. Open your favorite browser

All Chromium-based browsers (Google Chrome, Microsoft Edge,...), Firefox and Safari (not tested) are supported.

Internet Explorer is not supported but if you ran QRgen in IE it probably won't work properly.

4. Click on the menu icon ( or ) and click on the "Extensions"


5. Enable "Developer Mode"

Don't worry, you don't have to be a developer to do it.


6. Click the "Load unpacked" and select the "v1" foler

The "Load unpacked" button will open file explorer dialog.

7. Done.

The extension will appear in the extension section.
