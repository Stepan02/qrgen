How do I install the extension?

Wondering how to add an extension to your browser? Here's how:

The extension is not available at any browser extension marketplace yet
Therefore it must be downloaded from GitHub repository. You can find the latest release here.

1. Install ZIP file

First you must do is download ZIP file using the button on the top of this site or button below.

2. Export ZIP folder

Once you have downloaded ZIP file, you have to export it - right-click the file and click "Export all".

3. Open your favorite browser

All Chromium-based browsers (Google Chrome, Microsoft Edge,...) and Firefox are supported. Please check known issues for more information about Safari browser support. Internet Explorer is not supported.

4. Follow the instructions for your browser

Unfortunately, every browser works differently, so here are links to the documentation for instructions on how to add the extension.
