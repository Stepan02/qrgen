How do I install the extension?

Wonder how to use extension? Don't worry!


1. Install ZIP file

First you must do is download ZIP file using the button on the top of this site or button below.


2. Export ZIP folder

Once you have downloaded ZIP file, you have to export it - right-click the file and click "Export all".


3. Open your favorite browser

All Chromium-based browsers (Google Chrome, Microsoft Edge,...), Firefox and Safari (not tested) are supported.

Internet Explorer is not supported.


4. Follow the instructions for your browser

Unfortunately, every browser works differently, so here are links to the documentation for instructions on how to add the extension.
