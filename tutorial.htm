How do I install the extension?

Wondering how to add an extension to your browser? Here's how:

The extension is not available at any browser extension marketplace yet
Therefore it must be downloaded from GitHub repository. You can find the latest release here.

1. Download ZIP file

First, download the ZIP file using the button on the top of this site or button below.

2. Extract ZIP folder

Once you have downloaded the ZIP file, extract it:

  • Windows: right-click the ZIP file and select Extract All
  • macOS: double-click the ZIP file to extract it
  • Linux: use unzip

3. Open your favorite browser

Chromium-based browsers (Google Chrome, Microsoft Edge, Opera, ...) and Firefox are supported. Please check known issues for more information about Safari browser support. Internet Explorer is not supported.

4. Load the extension

Chromium-based browsers (Google Chrome, Opera, Microsoft Edge, ...)

  1. Open the extension settings page - chrome://extensions/ (or edge://extensions/ etc.) - or via your browser settings
  2. Enable Developer mode
  3. Click "Load unpacked"
  4. Select the extracted folder (not the downloaded file itself)
  5. Enjoy the extension ðŸ˜Š

Firefox

  1. Open about:debugging#/runtime/this-firefox page
  2. Click "Load Temporary Add-on"
  3. Select the manifest.json file from the extracted extension folder
