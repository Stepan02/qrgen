QRgen

The pocket QR code generator.

Generate QR code for anything - website, text, number, your info,... anywhere, anytime.

Check the FAQ if you have questions. Contact me if you don't see answer to your question, found a bug or you have problem. I appreciate the bug report, thanks!


Simple

QRgen is simple and small extension that works in all Chromium based browsers.

FAQ

  1. Is possible to save the QR code?
    1. Of course! You can save the QR code as an image and share it!
  2. When is next version out?
    1. I don't know. I'll update QRgen when I have some new features and bug fixes so I cannot say "update will be every month" or something.
  3. How do I update to current version?
    1. Sadly, you have to re-install the extension but is possible that in the future will the extension be on extension stores so it will get updated automatically.
  4. My QR code didn't loaded yet, why?
    1. QR code generating time depends on the amount of data to generate, so, if you are generating QR code of few words, the QR code will appear almost instantly but if you send around two hundred characters, it might take longer.
    2. Just wait few second.
  5. Can I download old versions?
    1. I'm sorry but you can't. It is because these old versions can have bugs and are unsupported.
  6. What is the character limit in a text field?
    1. I set a limit to 1 000 characters to avoid API problems. It probably won't happen that you need to enter so much data into the QR code, so the limit won't affect you much. Maybe in future I'll increase this limit but it depends on API request limits (how many times you can request data and how much data you are allowed to insert into the request).
