QRgen

The pocket QR code generator.

Generate a QR code for anything - website, text, number, your infomation,... anywhere, anytime.
If you have any questions, check out the FAQ.


Simple

QRgen is simple and small extension that works in all Chromium based browsers.


Supported by

  


Pocket

Extension is accessible due to one-click run so you can access it whenever you want.


Free

Yes, QRgen is free to use. You don't need to pay for this extension.


Open source

You can see QRgen source code on GitHub.


Download

QRgen is free to use, but there's a catch - you have to install it manually. It's not available on any extension marketplace, as publishing the extension is not worth it just yet. Instead, it's published as a GitHub release. Since you can't install it from the marketplace, you have to sideload it into your browser - don't worry, here is a tutorial.

You can visit the GitHub page by clicking the download button below.

Please note that the extension is desktop-only.


FAQ

Is it possible to save the QR code? Of course! You can save the QR code as an image and share it! You can download it by clicking the download button, or copy it by clicking on the image.

Do I need an internet connection? Yes. The extension generates QR codes using the goQR API, so you need an internet connection. However, client-side (offline) generation is planned for future release.

What types of QR codes can I generate? Here is the list:
  • URL - https://example.com
  • Phone number - tel:123456789
  • E-Mail - mailto:email@example.com

Why can't I generate a QR code with certain protocols? QR codes are a common attack surface for bad actors - a QR code using a special URL protocol may be used to deliver malicious code. While the risk typically applies to scanning suspicious or unknown QR codes, I want to ensure that this extension is not used to generate such potentially dangerous QR codes.

The following protocols are blocked:

  • javascript:
  • data:
  • vbscript:
  • file:
Learn more about QR code security

When is next version coming out? I don't know. I'll update QRgen when I have some new features and bug fixes, so I can't say "there will be an update every month" or anything.

How can I update to the current version? Unfortunately you have to reinstall the extension manually. The extension is not on any marketplace at the moment, but it might be there in the future.

My QR code hasn't loaded yet, why? QR code generation time depends on the amount of data you generate, so if you generate a QR code of a few words, the QR code will display almost instantly, but if you send around 200 characters and more, it may take longer. Just wait a few seconds.

Can I download older versions? You can download them from GitHub but they are not supported, may contain bugs and lack some features.

Why is character input limited to 2,000 characters? I have set the limit to 2,000 characters to avoid problems with the API (QRgen uses the goQR API). Generating a QR code with a large amount of data also negatively affects the readability of the image. It probably won't happen that you need to enter that much data in the QR code, so the limit won't affect you too much. I may increase this limit in the future, but it depends on the limits for API requests (how many times you can request data and how much data you are allowed to put in the request).

I want to suggest a feature, how can I do it? You can check the contribution guidelines on GitHub. Any suggestion is greatly appreciated!

I found a bug, how can I report it? If you found a bug, please report it on GitHub using the "Bug report" issue template.
Do not create issues or PRs related to security vulnerabilities. Report them here.
