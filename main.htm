QRgen

The pocket QR code generator.

Generate QR code for anything - website, text, number, your info,... anywhere, anytime.

Check the FAQ if you have questions. Create an issue on GitHub if you found a bug or you have problem. I appreciate the bug report, thanks!


Simple

QRgen is simple and small extension that works in all Chromium based browsers.


Supported by

    


Pocket

Extension is accessible due to one-click run so you can access it whenever you want.


Free

Yes, QRgen is free to use. You don't need to pay for this extension.


Open source

You can see QRgen source code on GitHub.

FAQ

  1. Is possible to save the QR code?
    1. Of course! You can save the QR code as an image and share it!
  2. When is next version coming out?
    1. I don't know. I'll update QRgen when I have some new features and bug fixes, so I can't say "there will be an update every month" or anything.
  3. How can I update to the current version?
    1. Unfortunately you have to reinstall the extension manually. The extension is not on any marketplace at the moment, but it might be there in the future.
  4. My QR code hasn't loaded yet, why?
    1. QR code generation time depends on the amount of data you generate, so if you generate a QR code of a few words, the QR code will display almost instantly, but if you send around 200 characters and more, it may take longer.
    2. Just wait a few seconds.
  5. Can I download older versions?
    1. Sorry, but you can't. This is because these older versions may contain bugs and are not supported.
  6. What is the character limit in the text box?
    1. I have set the limit to 1,000 characters to avoid problems with the API. It probably won't happen that you need to enter that much data in the QR code, so the limit won't affect you too much. I may increase this limit in the future, but it depends on the limits for API requests (how many times you can request data and how much data you are allowed to put in the request).
